# 接口

## **1. **根据用户id查询群组列表

> 根据用户 id 获取其所在的群组列表
> 根据用户id查询有没有加入的群组，没有的话返回公共群组\(默认是公共配置\)，有的话返回群组列表\(1个或多个\)

```yaml
URL：http://localhost:8088/v1/user/findGroup
Method：GET

```

#### 请求参数

|**参数**|**类型**|**描述**|
|---|---|---|
|`userId`|String|用户工号|

#### 请求示例

```yaml
userId: x30021831
```

#### 成功响应

```java
{
    "code": 0,
    "success": true,
    "msg": "success",
    "total": 2,
    "data": [
        {
            "id": 1,
            "name": "公共群组"
        },
        {
            "id": 2,
            "name": "可信院"
        },
        ...
    ]
}
```

#### 错误响应

```json
{
    "code": -9,
    "success": false,
    "msg": "获取用户群组列表失败",
    "data": null
}

```

## **4. **管理员/群组管理员添加群组

> 管理员（admin/group\_admin）添加群组

```java
● URL：http://localhost:8088/v1/admin/insertGroup
● Method：POST

```

#### 请求参数

|**参数**|**类型**|**描述**|**备注**|
|---|---|---|---|
|name|String|群组名称||
|config|json|群组配置|非必填|
|userList|list|用户列表|非必填|
|userList.userId|String|用户工号||
|userList.role|int|用户角色（0：成员；1：群组admin）||

#### 请求示例

> 可以仅传递部分请求参数

```java
{
    "name": "群组名称",
    "config": {
        "xxx10": "yyy10",
        "xxx20": "yyy20",
        ...
    },
    "userList": [
        {"userId": "x30021831", "role": "1"},
        {"userId": "y30034860", "role": "0"}
        ...
    ]
}

```

#### 成功响应

```json
{
    "code": 0,
    "success": true,
    "msg": "success",
    "data": null
}
```

#### 错误响应

```json
{
    "code": -9,
    "success": false,
    "msg": "添加群组失败",
    "data": null
}

```

## **6. **管理员/群组管理员修改群组配置

> 管理员（admin/group\_admin）根据群组id修改群组配置
> admin可以修改所有群组，group\_admin只可以修改自己管理的群组

```java
● URL：http://localhost:8088/v1/admin/updateGroup
● Method：POST

```

|**参数**|**类型**|**描述**|备注|
|---|---|---|---|
|group\_id|int|群组id||
|name|String|群组名称||

|userList.role|int|用户角色（0：成员；1：群组admin）||
